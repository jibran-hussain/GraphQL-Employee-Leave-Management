
## Installation

1 . Go the project directory.
```bash
  cd path-to-the-project
```

2 . Install dependencies
```bash
  npm i
```
3 .Specify the following environment variables:

  ```bash
PORT=3000
SALT_ROUNDS=10
JWT_SECRET_KEY='thisisasecretkey'
API_VERSION=v1

  ```
4 . Run the project using the following command:
```bash
  npm start
```
